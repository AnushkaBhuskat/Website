Title:

Develop a JavaScript to Display Today‚Äôs Date.

Software/Hardware Required:

  • Software: Web Browser (Chrome, Firefox, Edge), Code Editor (VS Code, Notepad++, Sublime Text)
  • Hardware: Computer with a web development environment

Theory:

JavaScript provides the Date object, which allows developers to fetch and manipulate date and time.

Key JavaScript Methods for Date Handling:

  • new Date() ‚Üí Creates a new date object representing the current date and time.
  • .getDate() ‚Üí Gets the day of the month (1-31).
  • .getMonth() ‚Üí Gets the month (0-11, where January is 0).
  • .getFullYear() ‚Üí Gets the four-digit year.

Using these methods, we can display today‚Äôs date dynamically.

Code:

<!DOCTYPE html>
<html>
<head>
   <title>Today's Date</title>
   <script>
      function showDate() {
         var today = new Date();
         var formattedDate = today.toLocaleDateString();
         document.getElementById("date").innerHTML = "Today's Date: " + formattedDate;
      }
      window.onload = showDate;
   </script>
</head>
<body>
   <h2>Today's Date</h2>
   <div id="date"></div>
</body>
</html>
