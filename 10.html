Practical No. 10 - Write a Program Using Arduino IDE for Blinking LED

Software/Hardware Required:

  • Arduino IDE
  • USB Driver, Arduino Board, LED, 220Œ© Resistor, Jumper Wires, Breadboard

Theory:

Basic LED blink program to test the Arduino board functionality using digitalWrite() and delay().

Circuit Diagram - Connections:

  • Pin 13 ‚Üí LED (Anode +)
  • GND ‚Üí LED (Cathode -) via 220Œ© resistor

Code:

// Blink LED on Pin 13
void setup() {
pinMode(13, OUTPUT);
}
void loop() {
digitalWrite(13, HIGH);
delay(1000);
digitalWrite(13, LOW);
delay(1000);
}
